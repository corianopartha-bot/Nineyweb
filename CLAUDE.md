# 项目：Niney · 蒲钰娇 个人名片网站

## 一、这是什么
- 一句话定位：一个让别人「一眼记住你」的个人名片网站，主人公是 Niney（蒲钰娇）
- 成功长什么样：陌生人打开后，能在 30 秒内感受到你的气质、看懂你是谁、
  看到你的作品/文字，并知道怎么联系你

## 二、技术与约定
- 技术栈：纯 HTML + CSS + 原生 JS，单页（single page），不上任何框架、不需要构建
- 代码约定：
  - 文件结构简单：index.html / styles.css / effects.js / assets/
  - 颜色、间距等用 CSS 变量统一管理，方便整体调
  - 引入任何新依赖（库、字体 CDN 等）前，先问我
- 运行 / 测试方式：直接双击 index.html 打开；或本地起 `python3 -m http.server` 预览

## 三、风格 / 人设
- 视觉：暗色底 + 玻璃质感（毛玻璃卡片、半透明、模糊）+ 一点超现实主义
- 语气：克制、有质感，不浮夸、不堆砌
- 红线：只用真实信息，绝不编造经历/数字/作品；没有的内容一律标【待补】

## 四、关键文件
- index.html：页面结构与所有文字内容
- styles.css：全部样式（玻璃质感、配色、响应式）
- effects.js：超现实交互效果（背景动效、About 视频随滚动浮现等）
- assets/：图片、视频等素材（含 About 背景视频）

## 五、铁律（护栏 · 必须遵守）
- 一次只改一个板块/功能；大改之前先给我方案，我同意再动手
- 不删、不改我没要求动的东西
- 拿不准就先问，不要自作主张
- 真实信息才放上去，没有就标【待补】，绝不编造

## 六、完成标准（那把尺子 · 逐条可检查）
1. 四个板块齐全：About / Work 作品 / 笔记 / 文章，顶部导航能点击跳转
2. 手机端（宽 375px）打开：不横向滚动、文字不溢出、布局不变形
3. 桌面端（宽 1280px+）：布局正常、留白舒服
4. 风格统一：全站共用一套配色和玻璃卡片样式，不花
5. 玻璃质感 + 超现实感可见（毛玻璃 + 至少一个动态/超现实元素）
6. 纯静态、无重型依赖，首屏内容秒出
7. 有可用的联系方式（如邮件链接能点）
8. 浏览器控制台无报错
9. 所有未提供的真实内容都用【待补】标注，无编造数据
10. About 视频背景：随点击/滚动有「从无到有」的浮现效果；若视频本身含
    「侧面→正面」画面，则滚动进度与画面进度联动（视频内容不支持则只做浮现，
    绝不编造画面）
- 自检不过就自己重做，最多 3 次；还不过就停下来报告卡在哪

## 七、怎么和我配合
- 每个任务：先列方案 → 我确认 → 你动手 → 自检 → 报告结果
- 报告里写清：动了哪些文件、为什么、怎么验证没坏

- 根目录维护 CHANGELOG.md，每个版本一段（功能 / 改进 / 修复 + 日期）
- 每次迭代主动问我要不要升版本，我确认后才升号 + 写日志
